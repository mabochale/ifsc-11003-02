      Progressive Enhancement

      it is defined as a strategy for web design that emphasizes core webpage content first, before adding advanced features and enhancements it is important to make sure that a website is perfectly compatible for all users, whether they are using a modern browser or an older one, or whether they have disabilities that require assistive technologies.

      to me, its like a steak dinner, where you start with the basic ingredients and then add spices and sauces to enhance the flavor without compromising the core taste.
